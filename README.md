# rn: mass rename done right


### features:
+ ultra minimal
+ supports regex and glob
+ shallow (default) and recursive modes
+ hidden files safe
+ collision/overwrite safe


### examples:
```bash
rn "&" and
rn -r this
rn --pattern "\s+" -
rn -p -d *-copy file
```


### installation:
```bash
nimble install rn
```
